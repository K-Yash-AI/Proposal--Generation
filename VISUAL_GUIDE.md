## Colour Scheme

| Element | Colour Code | Usage |
|---------|-------------|-------|
| Primary Accent | `#5B2D8F` (Purple) | Headings, table headers, box numbers |
| Background | `#E8E0F0` (Light Purple) | Table alternating rows, highlights |
| Text | `#2D2D2D` (Dark Grey) | Main body text |
| Secondary Text | `#7F7F7F` (Light Grey) | Captions, metadata |
| White | `#FFFFFF` | Table header text, spacing |

## Typography Hierarchy

```
H1 (Main Section Headings):       28pt, Purple, Bold
   └─ "Landscape & Objective Overview"
   
H2 (Sub-section Headings):        14pt, Dark Grey, Bold
   └─ "Year 1 Investment (AUD)"
   
H3 (Tertiary Headings):           12pt, Dark Grey, Bold
   └─ "Project Team Contacts"
   
Body Text:                         11pt, Dark Grey, Regular
   └─ Standard paragraph content
   
Captions/Labels:                   10pt, Light Grey, Italic
   └─ "CRM Workflow: End-to-end..."
```
