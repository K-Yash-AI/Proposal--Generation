# Quick Start Guide

## Setup (One-time)

### 1. Environment Variables
Create `.env` file in `/Users/khowal/proposal_agent/`:

```env
# Required: Claude API
ANTHROPIC_API_KEY=sk-your-key-here

# Required: Fireflies for transcripts
FIREFLIES_API_KEY=your-fireflies-key-here

# Optional: Google Drive (choose ONE)
# For Service Account (recommended):
GOOGLE_SERVICE_ACCOUNT_FILE=/path/to/service-account.json

# OR for OAuth2:
GOOGLE_CREDENTIALS_FILE=/path/to/credentials.json
GOOGLE_TOKEN_FILE=token.json

# Optional: Company Info (defaults already set)
COMPANY_NAME=Fruition
COMPANY_EMAIL=hello@fruition.com
```

### 2. Verify Installation
```bash
cd /Users/khowal/proposal_agent
source .venv/bin/activate
python -c "from agents.proposal_agent import ProposalAgent; print('✓ Ready')"
```

## Usage

### Method 1: Interactive CLI (Easiest)
```bash
source .venv/bin/activate
python run_agent.py
```

Follow prompts and enter:
- Client Name
- Company Name (optional)
- Email (optional)
- Transcripts count
- Upload to Drive? (yes/no)

### Method 2: Python Script
```python
from agents.proposal_agent import ProposalAgent

agent = ProposalAgent()
result = agent.run(
    client_name="Acme Corp",
    client_company="Acme Corporation",
    client_email="contact@acme.com",
    prepared_by="Your Name",
    transcripts_limit=5,
    upload_to_drive=True
)

print(result["local_path"])   # → /path/to/Proposal_Acme_Corp_20260330.docx
print(result["drive_link"])   # → https://docs.google.com/document/d/...
```

## Output

### Local File
- **Location**: `./proposals/Proposal_[ClientName]_[YYYYMMDD].docx`
- **Format**: Microsoft Word (.docx)
- **Editable**: Yes, in any application
- **Contents**: Fixed format with injected client-specific content

### Google Drive (If Uploaded)
- **Link**: Automatically shared
- **Access**: Public view (editable in Google Docs)
- **Version Control**: Automatic history

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No API key error | Set `ANTHROPIC_API_KEY` in `.env` |
| No transcripts found | Verify `FIREFLIES_API_KEY` and check Fireflies account |
| Drive upload fails | Ensure `GOOGLE_SERVICE_ACCOUNT_FILE` or `GOOGLE_CREDENTIALS_FILE` set |
| Module not found | Run `pip install -r requirements.txt` in venv |
| Python command not found | Run `source .venv/bin/activate` first |

## File Customization (Advanced)

After generation, you can:

1. **Edit in Word**: Open `.docx` file directly
2. **Edit in Google Docs**: Upload to Drive and edit in browser
3. **Convert to PDF**: Word → Export as PDF
4. **Extract text**: Open and copy content
5. **Update sections**: Edit specific proposal pages

## Tips

- ✅ Run the agent during business hours (API calls may be slower at night)

- ✅ Review the generated proposal before sharing
- ✅ Use Google Drive links for easy sharing with clients
- ✅ Save locally for backup before deleting from Drive
