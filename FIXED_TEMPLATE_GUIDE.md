# Proposal Agent - Fixed Format Template

## Overview

The proposal agent has been updated to use a **professional fixed-format template** for consistent, branded proposals. Every generated proposal now follows the exact structure and visual hierarchy you specified.

## Key Features

### 1. **Professional Fixed Format**
All proposals now include these standardized pages:

- **Cover Page**: Purple branding with client company and presenter info
- **Overview Boxes**: Three numbered sections (01, 02, 03) with key talking points
- **Landscape & Objective Overview**: Client situation and current challenges
- **Solution Design**: Solution approach with workflow diagram placeholder
- **Phase Details**: Detailed breakdown of each implementation phase
- **Project Deliverables & Timeline**: Table-based project schedule
- **Investment Summary**: Year 1 and ongoing costs with formatted tables
- **Next Steps**: Actionable items and contact information
- **Fruition Introduction**: Company boilerplate (fixed/non-editable)

### 2. **Claude-Generated Content Injection**
The Claude agent generates content for these sections dynamically:
- Understanding Your Needs (from transcripts)
- Proposed Solution (customized to client)
- Scope of Work (phased approach)
- Investment breakdown
- Next Steps

This content is automatically injected into the appropriate template locations.

### 3. **Google Drive Upload & Sharing**
```bash
# With optional Google Drive upload
upload_to_drive=True
```

When enabled:
- ✅ Document is automatically uploaded to Google Drive
- ✅ Made publicly shareable (with a unique link)
- ✅ Editable in Google Docs or downloadable as DOCX
- ✅ Share link returned immediately

## Usage

### Running the Agent

```bash
cd /Users/khowal/proposal_agent
source .venv/bin/activate
python run_agent.py
```

Follow the prompts:
```
Client Name (required): Acme Corp
Client Company (optional): Acme Corporation
Client Email (optional): contact@acme.com
Prepared By (optional): Your Name
Number of transcripts (default 5): 5
Upload to Google Drive? (y/n, default: n): y
```

### Programmatic Usage

```python
from agents.proposal_agent import ProposalAgent

agent = ProposalAgent()
result = agent.run(
    client_name="John Doe",
    client_company="Acme Corp",
    client_email="john@acme.com",
    prepared_by="Your Name",
    transcripts_limit=5,
    use_fixed_format=True,      # Always True for new proposals
    upload_to_drive=True         # Optional
)

# Result contains:
print(result["local_path"])   # Path to DOCX file
print(result["drive_link"])   # Shareable Google Drive link (if uploaded)
```


## Troubleshooting

### Issue: "No Google credentials configured"
**Solution**: Set either `GOOGLE_SERVICE_ACCOUNT_FILE` or `GOOGLE_CREDENTIALS_FILE` in `.env`

### Issue: Proposal saved locally but not uploaded to Drive
**Solution**: Google Drive upload failed silently. Check:
1. Google credentials are valid
2. You have write access to Drive
3. The Drive API is enabled in your Google Cloud project

### Issue: Links in DOCX are not editable
**Solution**: This is expected - the DOCX is the master editable file. For real-time editing:
1. Upload to Google Drive (`upload_to_drive=True`)
2. Open in Google Docs via the share link
3. Edit directly in browser
